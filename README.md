# yugioh_card_detection
This project consists of detecting the types of cards from the Yu-Gi-Oh Card Game.

# The number of types of cards to detect will be only 3:
- Monster
- Spell
- Trap

# Dataset
- 81 images.
- 81 label files.
- labels made on labelImg
